# Hacker News MERN Assignment

A full-stack MERN application that fetches and displays Hacker News stories with authentication and bookmark functionality.

---

# Features

- JWT Authentication
- Login & Register
- Bookmark Stories
- Protected Routes
- Search Stories
- Profile Page
- Dark / Light Theme
- Responsive UI
- Infinite Load More Pagination

---

# Tech Stack

## Frontend
- React.js
- React Router DOM
- Axios
- React Toastify

## Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
